GalPRIME
========

**GalPRIME** is an efficient, multi-threaded suite for testing surface brightness profile
extraction on a set of images, using provided input parameters.
It is designed to run smoothly on any provided dataset, and can provide results
for bins of various parameters, such as mass, redshift,
and star formation probability.
